# API InovConnect

1. Instalar npm (node package manager).
2. Na pasta da API digitar npm install para instalar todos os pacotes.
3. Para rodar a API basta digitar npm run devStart.

Para a aplicação flutter funcionar corretamente é necessário (ainda) copiar o seu endereço IPv4 e colar no caminho (lib/http/webclient.dart)
na variável url.
